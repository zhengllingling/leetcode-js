# 衡量算法的两个维度
## 时间复杂度
### 执行算法所需要的时间
### 执行时间在不同的环境下和数据量下值的变化会很大，所以就使用大O表示法，T(n) = O(f(n))
### f(n)表示代码的执行次数之和，O表示正比例关系
### 大O符号表示法，并不是表示真实的算法执行时间的，而表示的是算法执行时间的增长变化趋势的

## 空间复杂度
### 执行算法所需要占用的内存空间
### 空间复杂度不是用来计算算法实际占用的空间的
### 空间复杂度是对一个算法在运行过程中临时占用存储空间大小的一个量度
### 反应的是一个趋势，S(n)表示
